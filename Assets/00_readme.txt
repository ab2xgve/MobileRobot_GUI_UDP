起動手順
　0. RobotUIData/configフォルダ内の
　　　・GainConfig.txt
　　　・BrakeGainConfig.txt
　　　・UDPConfig.txt
     を設定。詳細は同フォルダ内の 00_readme.txt で
　1. RobotUI.exeで起動
  2. Runをクリック
  3. 右のラジオボタンボックスからモードを選択
　　　・Stop     : 停止, ジョイスティックの入力を受け付けない状態(モード値 : 0)
      ・Torque   : トルク入力、RobotUIData/config/GainConfig.txt で設定されたゲイン(モード値 : 1)
　　　・Velocity : 速度入力、RobotUIData/config/GainConfig.txt で設定されたゲイン(モード値 : 2)
　　　・UDP Test : 未実装
　　　・Other    : 未実装（仕様）
  4. ジョイスティック操作
　　　・左：並進(x,y)方向の入力
            横がx(右方向が正), 縦がy(上方向が正)
　　　・右：姿勢の入力, 
            縦軸から左側なら正の入力、右側なら負の入力。
　　　　　　ジョイスティックの中心点からの距離に応じて入力の大きさが決まる。
　5.終了時は ESC キー の入力

送信値について
　Simulinkへと送信される値。以下の順で構成される 7 要素の配列。
　　1. モード値、共通指令がどの入力モードかを判別するための値
　　2. x軸方向の並進入力
    3. y軸方向の並進入力
    4. z軸方向の並進入力
    5. x軸まわりの旋回入力
    6. y軸まわりの旋回入力
    7. z軸まわりの旋回入力
　値はすべてfloat型 = single型で構成されている。
　
受信値について
 Simulink から受信する値。以下の順で構成される 12 要素の配列。
　　1.  x軸方向の目標位置
    2.  y軸方向の目標位置
    3.  z軸方向の目標位置
    4.  x軸まわりの目標姿勢
    5.  y軸まわりの目標姿勢
    6.  z軸まわりの目標姿勢
    
　　7.  x軸方向のロボット位置
    8.  y軸方向のロボット位置
    9.  z軸方向のロボット位置
    10. x軸まわりのロボット姿勢
    11. y軸まわりのロボット姿勢
    12. z軸まわりのロボット姿勢

　値はすべてfloat型 = single型しか受け取れない。

ブレーキボタンについて
  ロボットを停止させるためのブレーキ、
　一度押すと赤色に変化、赤色の間は、偏差フィードバックし続ける。
　また、その間はジョイスティックからの入力も受け付けない。
　もう一度押すことで解除される。

　※付け焼刃のため調整不足、長時間使用すると発散するため使用は最小限
